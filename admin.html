ℹ️ 首次使用说明

1. 创建 GitHub Token：

访问 GitHub Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token。

勾选 repo 权限，生成后复制 token。

2. 设置密码：

首次使用时，输入你想设置的密码。密码的哈希值会存储在浏览器中。

3. 数据安全：

Token 和密码都存储在浏览器 localStorage 中，不会上传到任何服务器。
