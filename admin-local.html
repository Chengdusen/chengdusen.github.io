🚀 部署到 GitHub Pages

1

将下载的 data.json 覆盖项目中的 data/data.json

2
放入文件

将实际的安装包/文档放入 files/ 目录中

3
提交并推送

git add .
git commit -m "更新工具库"
git push origin main

4
等待部署

GitHub Pages 会自动部署，通常 1-2 分钟生效。
访问 https://chengdusen.github.io/ 查看。

📦 大文件处理（>50MB）

大于 50MB 的文件不适合直接放入仓库。建议使用 GitHub Releases：

  1. 在 GitHub 仓库页面点击 Releases → Create a new release
  2. 填写版本号（如 v1.0），上传大文件作为附件
  3. 发布后，复制附件链接填入工具项的下载链接
  4. 存储方式选择 GitHub Releases
